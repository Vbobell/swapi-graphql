# swapi-graphql

project created to test graphql and consume api [swapi](https://swapi.co/).

## dependencies required

- [node](https://nodejs.org/en/)

## tools the project

- nest

## Instalation

```bash
npm install
```

## Run

```bash
npm start
```

## Run test

```bash
npm run test
```

## Run build

```bash
npm run build
```

## Access

```bash
localhost:8000 or 127.0.0.1:8000
```
